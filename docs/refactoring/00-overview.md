## Как использовать

1. Открыть новый чат в Cursor
2. Указать нужный файл фазы:
   - `docs/refactoring/02-templates.md` - работа с шаблонами
   - `docs/refactoring/03-modules.md` - продолжение разделения
3. Попросить выполнить задачи по чеклисту
4. По завершении — отметить выполненные пункты

## Зависимости между фазами
